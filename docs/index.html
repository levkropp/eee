eee

A tiny Windows utility that restarts Explorer with a hotkey. Hold the combo for 10 seconds — that's it.

Ctrl + Alt + Shift + E  for 10 seconds

Download GitHub

Features

Invisible

No tray icon, no window. Completely silent until you need it.

Starts at login

Installs as a scheduled task. Runs automatically, survives reboots.

Safe

10-second hold prevents accidents. Release early to cancel.

Tiny

~250KB single executable. No dependencies, no runtime, no installer.

Install

Download the exe and double-click it. That's it.

Or from the command line:

eee.exe install

To remove, open Settings > Apps and uninstall eee, or run:

eee.exe uninstall
